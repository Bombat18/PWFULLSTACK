1. <h1> - Heading Tag

    • Use cases:
      • Indicate the main heading of a page or section.
    • Best practices:
      • Use only one <h1> per page or section.
      • Use sequentially numbered heading tags (<h2>, <h3>, etc.) for subheadings.

  2. <p> - Paragraph Tag

    • Use cases:
      • Represent a paragraph of text.
    • Best practices:
      • Use <p> to wrap blocks of text, rather than manually adding line breaks.
      • Avoid nesting block-level elements inside <p>.

  3. <strong> or <b> - Bold Tag

    • Use cases:
      • Emphasize important or significant text.
    • Best practices:
      • Use <strong> for semantic importance and <b> for presentational purposes.
      • Avoid using bold just for aesthetic purposes without semantic meaning.
